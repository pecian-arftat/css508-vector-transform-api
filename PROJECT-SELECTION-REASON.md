Features: 
Supports square matricies: 4x4.
Vector transformation for rotation, scaling, and translation.
Vector transformation with Euler Angles.
Vector transformation with Quaternions.
Vector transformation with inverse.
Dot Product
Cross Product


Reason
Before joining the GCSDD program, I began teaching myself graphic programming, which I continue to pursue. For this course, I propose developing an API that simplifies 4D vector transformations through a single function call, making this essential graphics programming concept more accessible. I will build and test the API using methods from CSS508B, then apply it to my personal project: a game engine for rendering 3D graphics. This project will also help me reinforce my understanding of linear algebra.
